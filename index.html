Los metodos/verbos de petición son para indicar la acción que se desea realizar para un recurso determinado, algunos de los metodos son;
método GET, método Head, método POST, el modo PUT, el método DELETE, el método connect, el método OPTIONS, el método TRACE y método PATCH

Elegir una de las siguientes temáticas: autos, videojuegos, cosmética, indumentaria.
El formulario debe solicitar los siguientes datos: nombre y apellido* , correo electrónico*, fecha de nacimiento, DNI*, teléfono, usuario de X, usuario de Instagram, ciudad, provincia.
